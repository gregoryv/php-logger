README
======

Logger module for basic logging in php.

* [API reference](https://gregoryv.github.io/php-logger/api)

Usage
-----

Add to your composer

  {
    "repositories": [
      {
        "type": "vcs",
        "url": "https://github.com/gregoryv/php-logger.git"
      }
    ],
    "require": {
      "gregoryv/php-logger": "*"
    }
  }
